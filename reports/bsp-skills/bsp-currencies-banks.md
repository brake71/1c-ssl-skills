**Дополнительно сверено:**

- Подтверждено **отсутствие общих модулей** `Валюты`, `Банки`, `Организации` (без суффиксов) — в БСП 3.1.11 такие модули не существуют. Реальные имена: `РаботаСКурсамиВалют`, `РаботаСБанками`, `ОрганизацииСервер`. Эта ловушка зафиксирована в Anti-patterns скила.
- Подсистема `Организации`: стабильный API (`ОрганизацииСервер`) и переопределяемый модуль (`ОрганизацииПереопределяемый`) — **разделены**. `ПриОпределении*` процедуры из `ОрганизацииПереопределяемый` предназначены только для замещения, не для вызова из прикладного кода.
- Подсистема `Банки` vs модуль `РаботаСБанками`: имя модуля **не совпадает** с именем подсистемы. Подсистема `Банки.xml`, модуль `РаботаСБанками.bsl` (без префикса, по аналогии с другими подсистемами БСП).
- `СформироватьСуммуПрописью`: 5 параметров (`СуммаЧислом`, `Валюта`, `БезДробнойЧасти`, `КодЯзыка`, `ДробнаяЧастьПрописью`) — сверено с `src/cf/CommonModules/РаботаСКурсамиВалют/Ext/Module.bsl:89`. Имена параметров дословно.
- `СведенияБИК`: 3 параметра (`БИК`, `КоррСчет`, `ТолькоАктуальные`) — сверено с `src/cf/CommonModules/РаботаСБанками/Ext/Module.bsl:41`. Первые два — `Знач`, третий — нет (передача по ссылке для булевого значения). Имена и порядок дословно.
- `ПересчитатьВВалюту` vs `ПересчитатьПоКурсу` — разные модули: первая в `РаботаСКурсамиВалют` (сервер, сам получает курсы по дате), вторая в `РаботаСКурсамиВалютКлиентСервер` (Клиент+Сервер, принимает готовые структуры курсов). Не путать.
- `СформироватьСуммуПрописью` — серверный метод, не UI. Для печатных форм идиоматичнее подключить источник данных через `РаботаСКурсамиВалют.ПодключитьИсточникДанныхПечатиЧислоПрописью` и использовать в макете — но это за рамками L3-скила, в §Anti-patterns скила дана отсылка к `bsp-print-reports`.
- `РаботаСКурсамиВалютСлужебный.ЗагрузитьКурсы` — присутствует в `#Область СлужебныйПрограммныйИнтерфейс` (строка 9), в скиле помечен ⚠️. Содержит ручную загрузку курсов; в прикладном коде запускается через `РаботаСКурсамиВалютКлиент.ПоказатьЗагрузкуКурсовВалют` (UI) или регламентным заданием.
- `ПолучитьДанныеКлассификатора` (Банки) — в `#Область УстаревшиеПроцедурыИФункции` (строка 142), заменён на `СведенияБИК`. В скиле помечен ⚠️ и в Anti-patterns.
