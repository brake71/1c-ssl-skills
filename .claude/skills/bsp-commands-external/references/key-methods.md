# Key methods (полный справочник)

Расширенный список методов подсистем **ПодключаемыеКоманды**, **ДополнительныеОтчетыИОбработки**, **СозданиеНаОсновании**. В основном `SKILL.md` — топ-20 самых частотных; здесь — оставшиеся 3.

| Метод | Сигнатура | Сервер/Клиент | Назначение | Пример вызова | Стабильность |
|---|---|---|---|---|---|
| `ПодключаемыеКоманды.ПараметрыРазмещения` | `ПараметрыРазмещения()` | Сервер | Конструктор структуры параметров размещения (`Источники`, `КоманднаяПанель`, `ПрефиксГрупп`, `ВладелецКоманд`) | `Парам = ПодключаемыеКоманды.ПараметрыРазмещения();` | стабильный |
| `ПодключаемыеКоманды.ПараметрыВыполненияКоманды` | `ПараметрыВыполненияКоманды()` | Сервер | Конструктор структуры `ПараметрыВыполнения` для серверной части команды (поля `ОписаниеКоманды`, `Форма`, `ЭтоФормаОбъекта`, `Источник`) | `Парам = ПодключаемыеКоманды.ПараметрыВыполненияКоманды();` | стабильный |
| `ДополнительныеОтчетыИОбработки.ЗагрузитьНастройки` | `ЗагрузитьНастройки(Ссылка)` | Сервер | Прочитать ранее сохранённые настройки обработки | `Настройки = ДополнительныеОтчетыИОбработки.ЗагрузитьНастройки(Ссылка);` | стабильный |
