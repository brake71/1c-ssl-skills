# Key methods (дополнение)

Дополнительный список методов подсистем **ЭлектроннаяПодпись** и **МашиночитаемыеДоверенности (ФНС)**. В основном `SKILL.md` — топ-15 самых частотных; здесь — оставшиеся 7. Это не полный справочник — для полного перечня запустите `python scripts/bsp_data_search.py module ЭлектроннаяПодпись --src <path/to/cf>`.

| Метод | Сигнатура | Сервер/Клиент | Назначение | Пример вызова | Стабильность |
|---|---|---|---|---|---|
| `ЭлектроннаяПодпись.ПроверитьСертификат` | `ПроверитьСертификат(МенеджерКриптографии, Сертификат, ОписаниеОшибки = Null, НаДату = Неопределено, ПараметрыПроверки = Неопределено)` | Сервер | Проверка только сертификата (без подписи) | `Верн = ЭлектроннаяПодпись.ПроверитьСертификат(Менеджер, Сертификат, Ошибка);` | стабильный |
| `ЭлектроннаяПодпись.СвойстваПодписи` | `СвойстваПодписи(Подпись, ПрочитатьСертификаты = Истина)` | Сервер | Извлечь свойства подписи (отпечаток, кем выдана, дата) | `Св = ЭлектроннаяПодпись.СвойстваПодписи(ДвоичныеДанныеПодписи);` | стабильный |
| `ЭлектроннаяПодписьКлиент.ДатаПодписания` | `ДатаПодписания(Оповещение, Подпись, ПривестиКЧасовомуПоясуСеанса = Истина)` | Клиент | Извлечь дату из данных подписи; результат — `Дата` или `Неопределено` в оповещении | `ЭлектроннаяПодписьКлиент.ДатаПодписания(Оповещение, ДвоичныеДанныеПодписи);` | стабильный |
| `ЭлектроннаяПодписьКлиент.ОткрытьНастройкиЭлектроннойПодписиИШифрования` | `ОткрытьНастройкиЭлектроннойПодписиИШифрования(Страница = "Сертификаты")` | Клиент | Открыть общую форму настроек подсистемы | `ЭлектроннаяПодписьКлиент.ОткрытьНастройкиЭлектроннойПодписиИШифрования();` | стабильный |
| `ЭлектроннаяПодписьКлиентСервер.РезультатПроверкиПодписи` | `РезультатПроверкиПодписи()` | Клиент + Сервер | Конструктор структуры результата проверки. Используется в `ПроверитьПодпись` с `ПараметрыПроверки.РезультатВВидеСтруктуры = Истина` | `Рез = ЭлектроннаяПодписьКлиентСервер.РезультатПроверкиПодписи();` | стабильный |
| `МашиночитаемыеДоверенностиФНСКлиент.ПроверитьДоверенность` | `ПроверитьДоверенность(Оповещение, Доверенность, ИдентификаторФормы = Неопределено)` | Клиент | Асинхронная проверка доверенности в реестре ФНС | `МашиночитаемыеДоверенностиФНСКлиент.ПроверитьДоверенность(Оп, МЧДСсылка);` | стабильный |
| `ЭлектроннаяПодписьСлужебный.ДоступнаЭлектроннаяПодпись` | `ДоступнаЭлектроннаяПодпись(ТипОбъекта)` | Сервер | Проверить, что у типа объекта есть реквизит `ПодписанЭП` (через `ОпределяемыйТип.ПодписанныйОбъект`) | `Если ЭлектроннаяПодписьСлужебный.ДоступнаЭлектроннаяПодпись(ТипЗнч(Док)) Тогда …` | ⚠️ служебный |

Для поиска методов/модулей в выгрузке конфигурации используйте python scripts/bsp_data_search.py method <имя>.
