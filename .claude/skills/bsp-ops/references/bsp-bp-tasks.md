
Скил по подсистеме **БизнесПроцессыИЗадачи** 3.1.11. Покрывает серверные и клиентские точки входа, через которые прикладной код **исполняет задачи**, **перенаправляет** их на других исполнителей, **принимает к исполнению / отменяет принятие**, **останавливает / активирует** бизнес-процессы, **получает форму** выполнения задачи и **запускает регламентный контроль** сроков. Также покрывает клиентскую обёртку, открывающую управляемую форму перенаправления списка задач.

Скил **не** описывает проектирование маршрутов, добавление прикладных бизнес-процессов и регистров адресации, настройку ролей и групп исполнителей — это задача архитектора конфигурации.

## When to use

- Нужно программно **выполнить задачу** от имени исполнителя — `БизнесПроцессыИЗадачиСервер.ВыполнитьЗадачу`.
- Нужно **перенаправить одну или несколько задач** новому исполнителю / в новую роль — `БизнесПроцессыИЗадачиСервер.ПеренаправитьЗадачи`.
- Нужно **открыть из списка задач** управляемую форму перенаправления — `БизнесПроцессыИЗадачиКлиент.ПеренаправитьЗадачи`.
- Нужно **принять** массив задач к исполнению или **отменить** принятие — `БизнесПроцессыИЗадачиСервер.ПринятьЗадачиКИсполнению` / `ОтменитьПринятиеЗадачиКИсполнению`.
- Нужно **остановить активный** бизнес-процесс или **возобновить** — `БизнесПроцессыИЗадачиСервер.ОстановитьБизнесПроцесс` / `СделатьАктивнымБизнесПроцесс`.
- Нужно **узнать параметры формы выполнения** задачи — `БизнесПроцессыИЗадачиСервер.ФормаВыполненияЗадачи`.
- Нужно **проверить, является ли задача ведущей** — `БизнесПроцессыИЗадачиСервер.ЭтоВедущаяЗадача`.
- Нужно **вызвать обработчик регламентного задания** `МониторингЗадач` или `УведомлениеИсполнителейОНовыхЗадачах`.

## Не использовать, если

- Нужны базовые утилиты общего назначения — `bsp-base-common`.
- Нужно открыть **список задач** текущего пользователя — это `ОткрытьЗначение`/`ОткрытьФорму`.
- Нужно **спроектировать маршрут** нового прикладного бизнес-процесса — это задача внедрения БСП в конфигурацию.
- Нужно **переопределить поведение** БСП по задачам — `БизнесПроцессыИЗадачиПереопределяемый`.
- Нужна **фоновая обёртка** для регламентной рассылки — `bsp-longs-and-jobs`.

## Core concepts

### Именование модулей

Подсистема обслуживается семейством модулей **с обязательным суффиксом**. ⚠️ **Модуля `БизнесПроцессыИЗадачи` (без суффикса) НЕ существует**.

| Модуль | Контекст | Назначение |
|---|---|---|
| `БизнесПроцессыИЗадачиСервер` | Сервер | **Стабильный API**: выполнение, перенаправление, принятие, остановка, активация |
| `БизнесПроцессыИЗадачиКлиент` | Клиент | **Клиентские обёртки** UI |
| `БизнесПроцессыИЗадачиВызовСервера` | Сервер | Серверные вызовы из клиентского кода |
| `БизнесПроцессыИЗадачиПереопределяемый` | Сервер | «Крючки» конфигурации (НЕ вызывать) |
| `БизнесПроцессыИЗадачиКлиентПереопределяемый` | Клиент | Клиентские «крючки» (НЕ вызывать) |

### ⚠️ Фантомы — НЕ вызывать

- ❌ `БизнесПроцессыИЗадачиСервер.ФормированиеЗадач` — **не существует**. Задачи создаёт сам бизнес-процесс.
- ❌ `БизнесПроцессыИЗадачиСервер.СоздатьЗадачу` — **не существует**.
- ❌ `БизнесПроцессыИЗадачиКлиент.ОткрытьФормуЗадачи` — **не существует**. Используйте `ОткрытьЗначение(...)`.

## Patterns

### 1. Выполнить задачу

```bsl
&НаКлиенте
Процедура ВыполнитьЗадачу(Команда)
    ТекущиеДанные = Элементы.СписокЗадач.ТекущиеДанные;
    Если ТекущиеДанные = Неопределено Тогда Возврат; КонецЕсли;
    ВыполнитьЗадачуНаСервере(ТекущиеДанные.Ссылка);
    ОповеститьОбИзменении(Тип("ЗадачаСсылка.ЗадачаИсполнителя"));
КонецПроцедуры
&НаСервере
Процедура ВыполнитьЗадачуНаСервере(ЗадачаСсылка)
    БизнесПроцессыИЗадачиСервер.ВыполнитьЗадачу(ЗадачаСсылка, Истина);
КонецПроцедуры
```

### 2. Перенаправление с предварительной проверкой

```bsl
ИнфоОПеренаправлении = Новый Структура;
ИнфоОПеренаправлении.Вставить("Исполнитель", НовыйИсполнитель);
// Шаг 1 — проверка
МожноПеренаправить = БизнесПроцессыИЗадачиСервер.ПеренаправитьЗадачи(
    МассивЗадач, ИнфоОПеренаправлении, Истина);
Если НЕ МожноПеренаправить Тогда Возврат; КонецЕсли;
// Шаг 2 — выполнение
ПеренаправленныеЗадачи = Новый Массив;
Успех = БизнесПроцессыИЗадачиСервер.ПеренаправитьЗадачи(
    МассивЗадач, ИнфоОПеренаправлении, Ложь, ПеренаправленныеЗадачи);
```

### 3. Остановка и возобновление БП

```bsl
БизнесПроцессыИЗадачиСервер.ОстановитьБизнесПроцесс(БизнесПроцессСсылка);
БизнесПроцессыИЗадачиСервер.СделатьАктивнымБизнесПроцесс(БизнесПроцессСсылка);
```

### 4. Получить и открыть форму выполнения задачи

```bsl
&НаКлиенте
Процедура ОткрытьФормуВыполнения(ЗадачаСсылка)
    ПараметрыФормы = БизнесПроцессыИЗадачиСервер.ФормаВыполненияЗадачи(ЗадачаСсылка);
    Если ПараметрыФормы = Неопределено ИЛИ ПараметрыФормы.Количество() = 0 Тогда
        ОткрытьЗначение(ЗадачаСсылка);
        Возврат;
    КонецЕсли;
    ОткрытьФорму("Задача.ЗадачаИсполнителя.ФормаВыполненияЗадачи", ПараметрыФормы, ЭтаФорма);
КонецПроцедуры
```

## Anti-patterns

### ❌ Вызывать модуль без суффикса

```bsl
// ❌ ОШИБКА: модуль БизнесПроцессыИЗадачи не существует
БизнесПроцессыИЗадачи.ВыполнитьЗадачу(ЗадачаСсылка);
```
```bsl
// ✅ Серверный модуль — БизнесПроцессыИЗадачиСервер
БизнесПроцессыИЗадачиСервер.ВыполнитьЗадачу(ЗадачаСсылка, Истина);
```

### ❌ Вызывать фантомные методы

```bsl
// ❌ ОШИБКА: методы не существуют
БизнесПроцессыИЗадачиСервер.ФормированиеЗадач(БизнесПроцесс);
БизнесПроцессыИЗадачиКлиент.ОткрытьФормуЗадачи(ЗадачаСсылка);
```

### ❌ Создавать задачу без бизнес-процесса

```bsl
// ❌ Задача без БП не пройдёт проверки
НоваяЗадача = Задачи.ЗадачаИсполнителя.СоздатьЗадачу();
НоваяЗадача.Исполнитель = ...;
НоваяЗадача.Записать();
```

### ❌ Вызывать серверный метод с тонкого клиента

```bsl
// ❌ БизнесПроцессыИЗадачиСервер недоступен с тонкого клиента
&НаКлиенте
Процедура Перенаправить()
    БизнесПроцессыИЗадачиСервер.ПеренаправитьЗадачи(...); // ОШИБКА
КонецПроцедуры
```

### ❌ Забыть `ТолькоПроверка = Истина`

```bsl
// ❌ Ошибка придёт после частичного изменения
Успех = БизнесПроцессыИЗадачиСервер.ПеренаправитьЗадачи(МассивЗадач, Инфо);
```


Для поиска методов/модулей в выгрузке конфигурации используйте python scripts/bsp_ops_search.py method <имя>.
