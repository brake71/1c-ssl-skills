# Key methods (полный справочник)

Расширенный список методов подсистем **ЗавершениеРаботыПользователей** и **УдалениеПомеченныхОбъектов**. В основном `SKILL.md` — топ-10 самых частотных; здесь — все остальные, сгруппированные по модулям.

## ЗавершениеРаботыПользователей — серверный API (дополнительные)

| Метод | Сигнатура | Назначение | Пример вызова |
|---|---|---|---|
| `СоединенияИБ.ПараметрыБлокировкиСеансов` | `ПараметрыБлокировкиСеансов(ПолучитьКоличествоСеансов = Ложь)` → `Структура` | Получить параметры текущей блокировки ИБ | `Парам = СоединенияИБ.ПараметрыБлокировкиСеансов(Истина); КолВо = Парам.КоличествоСеансов;` |
| `СоединенияИБ.НовыеПараметрыБлокировкиСоединений` | `НовыеПараметрыБлокировкиСоединений()` → `Структура` | Создать пустую структуру параметров для `УстановитьБлокировкуСеансовОбластиДанных` | `Парам = СоединенияИБ.НовыеПараметрыБлокировкиСоединений(); Парам.Начало = ТекущаяДата(); Парам.Сообщение = "Блокировка области";` |
| `СоединенияИБ.ПолучитьБлокировкуСеансовОбластиДанных` | `ПолучитьБлокировкуСеансовОбластиДанных(ПоМестномуВремени = Истина)` → `Структура` | Прочитать блокировку сеансов области данных | `Блокировка = СоединенияИБ.ПолучитьБлокировкуСеансовОбластиДанных();` |

## УдалениеПомеченныхОбъектов — переопределяемый модуль

|---|---|---|---|---|
| `ПередУдалениемГруппыОбъектов` | `ПередУдалениемГруппыОбъектов(Контекст, УдаляемыеОбъекты)` | Сервер | Крючок **перед** удалением группы (вне транзакции). Можно проинициализировать `Контекст` для передачи в `ПослеУдаления` | `УдалениеПомеченныхОбъектовПереопределяемый` |

| `ПриОпределенииОбъектовСКомандойПоказатьПомеченные` | `ПриОпределенииОбъектовСКомандойПоказатьПомеченные(Объекты)` | Сервер | Определить, в формах списков каких объектов показывать команды «Показать помеченные» и «Перейти к помеченным» | `УдалениеПомеченныхОбъектовПереопределяемый` |
